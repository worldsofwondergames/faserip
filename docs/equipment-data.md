## Weapon Tags

| Tag | Description |
|-----|-------------|
| oneHanded | Can be fired one-handed without penalty |
| twoHanded | Requires two hands |
| illegal | Possession is a misdemeanor |
| military | Possession outside military is a crime |
| powerPack | Operates from power pack |
| bursts | May affect multiple adjacent targets |
| scatters | Affects all within one area |
| twoMenToFire | Two persons required (one may fire at half rate, -2CS) |
| stationary | Does not move, but can turn |

## Attack Types

- **S** = Shooting
- **E** = Energy
- **F** = Force
- **EA** = Edged Attacks
- **ET** = Edged Throwing (Throwing Edged)
- **BA** = Blunt Attacks
- **BT** = Blunt Throwing (Throwing Blunt)
- **Gp** = Grappling
- **Gb** = Grabbing

## Vehicle Types

- **Road** - Craft for hard, flat surfaces (suffer penalties off-road)
- **Off-Road** - Land vehicles for rough ground
- **Railed** - Follow predetermined path/track
- **GEV** - Ground Effect Vehicle (hovercraft)
- **Air** - Aircraft of all types
- **Space** - Spacecraft capable of sustained space travel
- **Water** - Boats and ships
- **Sub** - Submarines and submersibles
